# TREConsulta
Helper tool for Logistical Support workers during Brazilian elections

## data-extraction
Holds transformation code to extract information from a pdf file and persist it in a proper manner such as a json file or a database.

## application
Holds Android application code. It is a very simple application that allows a faster and easier way to search by voter's information. It is not intended to be used by everyone but only by people working in Brazilian elections in a role called Logistical Support (Apoio Log√≠stico).
